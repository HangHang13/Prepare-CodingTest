- 알고리즘 준비는 멘탈관리가 중요하다

- 실력이 만들어지면 결과가 잘 나오며, 실력이 아직 안만들어졌다면, 결과가 잘 나오지 않는다.
- 내가 어느위치인지 알아야한다. 옆사람과의 비교는 필요없다.
